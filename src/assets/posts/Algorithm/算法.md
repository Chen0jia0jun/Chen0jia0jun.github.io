# Dijkstra(迪杰斯特拉)算法

需要的参数：

-源点vs

-终点f

-前驱数组prev[]

-源点到各个节点的距离数组dist[]

-这个图的点的数量q

```

}
```

算法步骤：

1.初始化

```
int q; //顶点数
int prev[],dist[];
void Dijkstra(int vs,int f){
	int i,j,k;
	int min,temp;
	int flag[q];
	for(i = 0;i<q;i++){
		flag[i] = 0;
		dist[i] = martix[vs][i];
		prev[i] = vs;
	}
	flag[vs] = 1; 
	dist[vs] = 0;
}
```

2.遍历q-1次，每次更新一个节点到源点的最短距离

2.1 在为更新的节点中，找到距离源点最近的一个点

2.2 标记找到的点k

2.3 已知顶点k到源点的最短路径后，更新没有到源点路径的点和不是最小路径的点

```
...

for(i = 1;i<q;i++){
	for(j = 1;j<q;j++){
	min = INF_MAX;
	if(!flag[j] && min>dist[j]){
		min = dist[j];
		k = j;
		}
	}
	
	flag[k] = 1;
	if(k==f) break; //抵达终点
	
	
	for(j = 0;j<q;j++){
		temp = (martix[k][j]==INF_MAX ? INF_MAX : (min+martix[k][j]));
		if(!flag[j] && temp<dist[j]){
			dist[j] = temp;
			prev[j] = k;
		}
	}
}
```



# 双指针

从暴力双重循环---->用双指针优化时间复杂度

**分类**

1.左右指针：两个指针，相向而走，中间相遇

2.快慢指针：两个指针，有快有慢，同向而行

3.灵活运用：两个指针，灵活运用，伺机而动

## 左右指针（对撞指针）

先从暴力出发，合理利用题目规则（比如有序或者求解的数的计算方式独特），减少遍历次数，降低时间复杂度；

- **核心思想：**通过左右指针的协同移动，缩小问题的搜索范围。利用数组的有序性，通过比较指针指向的值与目标值的关系，决定移动哪个指针。
- **典型应用：**
  1. **两数之和：**在有序数组中找到两个数，使它们的和等于目标值。
  2. **三数之和：**找到数组中三个数，使它们的和等于目标值。
  3. **回文串判断：**判断一个字符串是否为回文串。

## 快慢指针

解决链表的某些问题，可以把生活中的一些问题抽象成链表问题，再使用快慢指针

- **典型应用：**

  1.**链表环检测：**判断链表中是否存在环。

  2.**链表中点查找：**找到链表的中点。

  3.**滑动窗口：**解决子数组或子字符串的相关问题。

```
#define lowbit(x) (x&(-x))

void add(int x,int k){
	for(int i = x;i<=n;i+=lowbit(i)){
		tr[i] += k;
	}

int sum(int l,int r){ //返回区间l-r的数的和
	long long suml = 0,sumr = 0;
	for(int i =l-1;i>0;i--){ //计算出1-（l-1）的和
		suml += tr[i];
	}
	for(int i =r;i>0;i--){ //计算出1-r的和
		sumr += tr[i];
	}
	return sumr-suml;
}
```
